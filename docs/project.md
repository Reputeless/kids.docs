# Siv3D for Kids プロジェクト

## 概要

Siv3D for Kids の開発と、Siv3D for Kids を教材として活用するプログラミング体験活動を全国規模で展開します。これまでの活動実績については [イベント日程](https://siv3d-for-kids.github.io/events/) をご参照ください。
